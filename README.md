# Pizza_sales_MySql_Project
Pizza sales data analysis using MySQL to uncover sales trends, popular products, and ordering patterns through 6 analytical SQL queries.

In total, 13 analytical questions were solved using SQL to explore different aspects of the dataset. For the purpose of presentation, the six most impactful analyses were selected and showcased in the project PDF. These highlight key findings such as top-selling pizzas, peak order times, and revenue trends, demonstrating how SQL can be used effectively for real-world business data analysis.
